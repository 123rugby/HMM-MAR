void hidden\+\_\+state\+\_\+inference(gamma, Xi, B, Pi\+\_\+0, P)

A function to perform inference of hidden states in an H\+MM.

The outputs, gamma and Xi, need to be passed in with memory already declared.

gamma -\/ (n\+Samples -\/ order) x n\+Classes

Xi -\/ (n\+Samples-\/1-\/order) x n\+Classes$\ast$$\ast$2

scale -\/ (n\+Samples x 1)

The inputs are

B -\/ n\+Samples x n\+Classes

Pi\+\_\+0 -\/ 1 x n\+Classes

P -\/ n\+Classes x n\+Classes

order -\/ integer
